房产中介管理系统<br>
===
*房产中介管理系统是一个用于房产中介公司内部业务工作的系统，系统功能包括房源管理，客源管理，人事管理，以及权限管理。<br>
*项目使用了`Spring`+`SpringMvc`+`mybatis`+`MySQL`的架构，前端主要使用了`jQuery`，`bootstrap`。整合了`shiro`框架。<br>
